# NET.S.2018.Dementey.05
Fifth day .NET EPAM

### Задание 1
Разработать неизменяемый класс Polynomial (полином) для работы с многочленами степени от одной переменной вещественного типа (в качестве внутренней структуры для хранения коэффициентов использовать sz-массив).
Для разработанного класса
- ul переопределить виртуальные методы класса Object;
- ul перегрузить операции, допустимые для работы с многочленами (исключая деление многочлена на многочлен), включая “==” и “!=”.
- ul Разработать unit-тесты.

### Задание 2
Реализовать алгоритм “пузырьковой” сортировки непрямоугольного (jagged array) целочисленного массива **(методы сортировки класса System.Array не использовать)** таким образом, чтобы была возможность упорядочить строки матрицы:
- ul в порядке возрастания (убывания) сумм элементов строк матрицы;
- ul в порядке возрастания (убывания) максимальных элементов строк матрицы;
- ul в порядке возрастания (убывания) минимальных элементов строк матрицы;
- ul Разработать unit-тесты.
